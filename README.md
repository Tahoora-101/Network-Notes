# Network+ Compendium: A Structured Journey

This repository is my curated knowledge base for the CompTIA Network+ certification, built as part of my journey toward a Master's in Cybersecurity Research.

## My Approach

Rather than passive note-taking, I focus on:
- **Conceptual Clarity:** Using analogies (like the "Neighborhood" for IP subnets) to ground abstract concepts.
- **Visual Learning:** Creating and sourcing clear diagrams to illustrate complex systems.
- **Practical Connection:** Always linking theory to real-world security implications and troubleshooting.
